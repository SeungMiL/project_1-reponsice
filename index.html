자주 묻는 질문

  • 방에 입장이 안되고 계속 로딩 화면만 나옵니다.
    Step1. 쿠키나 캐시를 삭제하고 브라우저를 최신 버전으로 업데이트 해주세요. ※ 쿠키삭제방법 [ 아이폰 ] 설정 - Safari - 방문 기록 및 웹 사이트 데이터 지우기
    [ 안드로이드 ] 크롬 앱 실행 후 우측 상단(점3개) 터치 - 설정 - 개인정보 보호 - 인터넷 사용 기록 삭제 - 기간 : 전체기간 - 아래 항목 전체 체크 - 인터넷 사용 기록 삭제

    Step2. 크롬 확장 프로그램에 들어가서 adblock 등 block 하는 프로그램이 설치되어 있는 경우 삭제 해주세요. Step3. 다시 OHO!에 접속해주세요.

    Step4. 문제가 해결이 안 될 경우 크롬 브라우저에서 Ctrl+Shilft+I를 누르고 2번째 [ Console ] 메뉴 화면을 cs_maaltalk@dial070.co.kr로 보내주세요.

    Step5. 크롬 브라우저 외에 firefox, Safari 에서도 가능하니 해당 브라우저를 이용해 주세요.
  • OHO!와 스마트 펜은 어떻게 연결하나요?
    Step1. Window 장치 관리자에서 Bluetooth가 켜져 있는지 확인해 주세요.

    Step2. OHO! 접속 후 공유하기(필기 내용 공유, 파일공유) 화면에서 Pen 전원 버튼을 켜주세요.

    Step3. 오른쪽 하단 Pen connect 버튼을 클릭해 주세요.

    Step4. 왼쪽 상단 화면에서 검색된 스마트 펜을 페어링 해주시면 됩니다.
  • OHO!와 스마트 펜이 연결되지 않아요.
    Step1. Window 장치 관리자에서 Bluetooth가 켜져 있는지 확인해 주세요.

    Step2. Bluetoothr가 켜져 있다면 스마트 펜 전원을 켜주시고 파란불이 들어왔는지 확인해 주세요. 불이 들어오지 않으면 건전지를 교체하시고, 그래도 불이 들어오지 않으면 기기에 이상이 있을 수 있습니다.
  • 이메일이 인증되지 않습니다.
    메일함에 인증 메일이 없다면 스팸메일함을 확인해주세요. 미인증 계정으로 로그인 시 메일을 재발송 해드리오니 메일함을 재확인 해주세요. 그래도 메일이 오지 않으시거나 인증이 정상적으로 되지 않으실 경우 SeeUhelp@naver.com로 문의주세요.
  • 누가 회의를 녹화 할 수 있습니까?
    회의 주최자만 녹화 할 수 있습니다.
    ※ 녹화는 [ 최대 14일 ] 까지만 보관됩니다.
  • 회의에 인원들을 초대하려면 어떻게 합니까?
    회의방 입장 > 좌측 하단 Invite 버튼 클릭
    카카오톡으로 링크 전송: 1. 복사 버튼을 클릭하신후 카카오톡으로 링크를 전달해주세요.
    이메일로 링크 전송: 1. 이메일 입력 후 초대하기 버튼 클릭 2. 그룹메일로 초대할 경우 회의방 링크를 복사 후 본인 메일에서 직접 그룹메일로 보내주셔야 합니다
  • OHO!는 어떤 웹 브라우저를 지원합니까?
    Window PC: Chrome(권장), Firefox 및 Opera
    Mac: Safari (11 버전 이상)
    iOS : Safari (11 버전 이상)
    Android OS : Chrome (80 버전 이상)
    지원불가 : IE, Edge, 카카오톡 내장 브라우저, 네이버 브라우저 등
  • 엔터프라이즈 계약 절차가 어떻게 되나요?
    엔터프라이즈의 경우 간단한 서비스 사용계약 후 1개월 단위로 계약을 하게 됩니다. 계약은 전자계약 서면계약 모두 가능합니다.
  • 엔터프라이즈 요금제의 선생님 모드는 무엇인가요?
    현재 체험해볼 수 있는 OHO! for Education 기능입니다. 2시간 무료 체험이 가능합니다.
